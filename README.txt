QA Cinema Project v1.0.0 25/08/2017 ===============================================

User Instructions: ----------------------------------------------------------

Please install node.js v6.11.1 LTS from https://nodejs.org/en/

Please install mongoDB from https://www.mongodb.com/download-center#community

If using a Microsoft operating system, please open the command line. Open terminal if using macOS or Linux.

Direct yourself to the location of the root folder using the following in command line: CD Desktop/qaOrangeAngular/qaOrangeBackend

Please run 'npm install' in the command line/terminal to install all back-end node modules.


-----------------------------------------------------------------------------

In a separate command line window:

Direct yourself to the location of the public folder using the following command line: 'CD Desktop/qaOrangeAngular/qaOrangeBackend/frontend'

Please run 'npm install' in the command line/terminal to install all frontend node modules.

------------------------------------------------------------------------------
Finally:

Open Google Chrome.

Please install the Google Chrome extension named 'Allow-Control-Allow-Origin:*' by vitvad. Remember to turn off this extension when not viewing this cinema site.

Enter 'npm start', in the backend command line terminal. This will run 3 commands automatically which will launch the project and open up the homepage of the website.

Please ensure that the extension detailed above is on for the website.
